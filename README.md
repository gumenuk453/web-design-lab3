## Завдання
Адаптувати програмний код ЛР№2 до вимог фреймворка VueJS та забезпечити завантаження необхідних даних з Web-сервера за допомогою асинхронних запитів.

## Про проєкт "MyContacts"
Fullstack-додаток (телефонний довідник) з клієнт-серверною архітектурою.
* **Frontend:** Vue 3, Bootstrap 5. Реалізовано патерн MVC для управління контактами.
* **Backend:** Node.js, Express, SQLite. Налаштовано REST API для авторизації та CRUD-операцій з базою даних.
